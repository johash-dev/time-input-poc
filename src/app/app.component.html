Angular 15.2.3 TimeInput POC (Reactive Forms + NgRx)

TimeInputComponent spec
`app-time-input`
Angular 15 • ControlValueAccessor + Validator • 12h UI → emits 24h `HH:mm`
Reactive Forms NgRx-friendly Minute step Min time Picker + typing
What it guarantees
  • Reactive-forms compatible: implements ControlValueAccessor + Validator.
  • Output safety: it never emits invalid values to the form (invalid typing stays UI-only).
  • Normalized value: committed value is always HH:mm (24h) or null.
  • Two ways to enter: type or pick (both enforce minuteStep + minTime).
Commit & emit rules
  • Typing commits: on blur or Enter.
  • Picker commits: immediately on selecting hour/minute/AM-PM.
  • When invalid: shows an error and keeps the last committed value (no emit).
  • Clearing: empty input commits to null.
Open/close behavior
  • Only the clock icon toggles the panel (input area is for typing).
  • Does not auto-close on selection.
  • Closes on OK, CANCEL, Escape, or clicking outside the component.
Supported input formats (typing)
  • 12h: h:mm AM/PM (e.g. 2:30 PM)
  • 24h: HH:mm (paste convenience)
  • Dot format: h.mm (e.g. 2.30 PM) is accepted and normalized.
Validation rules
  • required: empty value invalid (form error required).
  • minuteStep: committed time must be divisible by step (e.g. 30 → :00 or :30).
  • minTime (minHour + minMinute): committed time must be \(\ge\) min; picker disables earlier options.
Error message behavior
  • Internal messages: shown for format / minTime / minuteStep while typing.
  • External override: if errorMessage is provided, it replaces internal messages.
  • MinTime copy: message shows “after previous slot” for step-based mins (start 10:30 → copy “after 10:30”).
API documentation
Inputs
label?: string
Field label shown above the input value.
placeholder?: string
Placeholder shown when empty.
widthConstraint: 'fixed' | 'fluid'
fixed keeps 170px max-width; fluid fills container (field + panel).
minuteStep: number / nzMinuteStep: number
Minute increment enforced for typing + picker (default 30).
minHour: number, minMinute: number
Minimum allowed time (24h); disables picker options and blocks commits below min.
required: boolean
Enables required validation (Validator returns { required: true } for empty).
requiredAsterisk: boolean
Shows the red asterisk (independent of validation).
showRedAsterisk: boolean
Compatibility flag used with required for showing the asterisk.
disabled: boolean
Disables typing + picker (merged with CVA disabled state).
errorMessage?: string
External error text that overrides internal validation messages.
showCancel: boolean
Show/hide CANCEL button in the picker footer.
Outputs
ngModelChange: EventEmitter<string | null>
Compatibility output that emits the committed HH:mm value (or null).
Forms integration (Reactive Forms)
  • Implements ControlValueAccessor and Validator.
  • Commits value on blur or Enter. Picker selection commits immediately.
  • Will not emit invalid values (prevents autosave on invalid input).
